# Business Task Manager

Monorepo with:
- apps/web (React/Next.js/Tailwind) — web app for monthly marketing task management.
- apps/mobile (Expo/React Native) — companion mobile client.

## Run web
cd apps/web
npm install
npm run dev
# open http://localhost:5173 or 3000 (depending on your dev server)

## Run mobile (Expo)
cd ../mobile
npm install
npx expo start

## Tech
React, Next.js/Vite, Tailwind CSS, Expo, TypeScript/JS, Lucide Icons
