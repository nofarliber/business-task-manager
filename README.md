
A full-stack web and mobile application designed to help business owners manage and track their monthly marketing activities.  
Each business type (Law Firm, Web Designer, Beautician, Online Store, or Fitness Instructor) receives a personalized dashboard with pre-defined tasks and progress tracking.


- Monthly marketing task planning tailored to business type.  
- Task status tracking with automatic progress updates.  
- Responsive, modern UI built with TailwindCSS.  
- Separate dashboards and task sets for different business categories.  
- Compatible with both web and mobile environments.
| Layer | Technologies |
|--------|---------------|
| Frontend | React, Next.js, TailwindCSS |
| Mobile | Expo, React Native |
| Backend | Node.js (Next.js API routes) |
| Tools | Git, GitHub, npm |
| Design | Responsive layout, reusable UI components |
